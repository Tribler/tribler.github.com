Towards Anonymity

Privacy in downloading

Governments are trying to control the media by blocking websites,hunting down bloggers and installation of Internet Kill switches. We are working on solving these issue by expanding our Open Source software.


Becoming "lawyer-proof" is our current target.


Legal threats and harassment are an important tool today to stop dissidents and unpopular opinions in the media. Tribler aims to fully protect you from harm, but beware that we're not finished yet.

Disclaimer

Tribler has come a long way in the field of file sharing and aims to enrich the Tribler community with anonymous downloading. The protocols and methods that are being used have proven to be very effective. Unfortunately it is not yet possible to download regular files anonymously.


Currently we are testing the new version and optimizing the downloading performance. Feel free to join this process by downloading the newest version. In this version it is possible to download a test file anonymously, yet there is no absolute guarantee that the downloading performed by Tribler will be 100% anonymous.


How it works

Bittorrent Swarm

When you search for content you send out a request to all Tribler users. These users are connected in the Tribler community. You will receive the content from one of the other members. This can be traced.

First proxy layer

Via a proxy layer the content is passed to you. Therefore the direction between you, the receiver, and the sender is interrupted. Proxies are virtual places that forward the content as they are told.

Second proxy layer

Adding a second layer of proxies makes it even more difficult to trace the route of the content, creating anonymity for you as a user.

The hidden provider

Triblers future steps include making the provider hidden as well. When this is possible, uploading can become just as anonymous as downloading. This will be achieved by adding proxy layers for the provider side, just as the above-mentioned steps 2 and 3.

The Tribler 6.3.0 Anonymity test

In the newest release of Tribler software an Anonymity test has been included. The current version incorporates the first three of the above mentioned steps, but the next release will offer also anonymous uploading.

In this release it is possible to anonymously download a test file within the Anonymity tab. Be aware that no other files can be downloaded anonymously yet, but after the trial this should be feasible soon.
