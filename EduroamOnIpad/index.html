eduroam uses not so common authentication methods (WPA2 Enterprise with PAP over TTLS). In an Apple iPad, iPhone or iPod Touch is not possible to configure that directly from the device, so you need a profile configuration file created with the iPhone Configuration Utility (available on MacOS X: ​http://support.apple.com/kb/DL851; and Windows: ​http://support.apple.com/kb/DL926).

Attached is a generic configuration file for eduroam at TU Delft. To use it, just open it from the device you want to configure, turn the new profile on and introduce your [netId]@tudelft.nl and your password.
