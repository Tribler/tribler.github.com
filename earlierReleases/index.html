New in Tribler 4.5

  • Major reworking of the code, introducting a well-defined API to control the download engine, and a new database layer. Also includes SwarmPlayer and live streaming code.

New in Tribler 4.1

  • Family filter
  • Keyword search improved (now searching in the Tribler network as well)
  • Improved download performance
    • mainline DHT
    • uTorrent Peer Exchange support
  • Top 10 Tribler uploaders in profile view
  • List view for discovered files and discovered persons
