How to Add a New Wiki Page

These are the steps to create a new wiki pages:

  1. Choose a name for your new page. See WikiPageNames for naming conventions.
  2. Edit an existing page, and add a hyperlink to your new page. Save your changes.
  3. Follow the link you created to take you to the new page. Trac display a "describe PageName here" message.
  4. Click the "Edit this page" link to edit and add content to your new page
  5. All done. Your new page is published.

See also: TracWiki, WikiFormatting, TracLinks, WikiDeletePage
