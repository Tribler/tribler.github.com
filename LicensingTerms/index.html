Tribler Licensing Terms

Tribler is licensed under a GNU Lesser General Public License (LGPL) v2.1 license. As it is built on the ABC, BitTornado and BitTorrent clients, some parts of the code still fall under the original licenses. Which code falls under which license is determined by the authors listed at the top of each source file. ​See the official text of our license

Tribler binaries have an extended license that includes the licenses of all the open-source software that we ship along in our installers. ​See the official text of our license for binaries.
