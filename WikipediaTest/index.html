Wikipedia Test Sep 2010

Please install SwarmPlayer from ​http://swarmplayer.p2p-next.org and then find a new (=not listed below) Wikipedia page with a video on it, and play the video using the SwarmPlayer. To do that you must add ?withJS=MediaWiki:MwEmbed.js to the URL, e.g. ​http://en.wikipedia.org/wiki/Polar_bear?withJS=MediaWiki:MwEmbed.js. Alternatively, if you have a Wikipedia account, you can enable âmwEmbed supportâ in Preferences/Gadgets. There is a visual check if this worked: if so, there is a big play button in the middle of the video (and the SwarmEngine should appear in the systray).
